# 基本的数据结构 - 集合
## 集合的概念特点
> 1. 无序
> 2. 元素不能重复
> 3. 是一种特殊的数组，因为无序，所以无法通过下标访问元素 
> ## 集合常用的方法
> 1. add(element): 添加新元素到集合尾部
> 2. delete(element): 删除集合的第一项元素，并将其返回
> 3. has(element): 集合存在指定的元素返回true,反之false
> 4. values():返回一个包含集合中所有值的数组
> 5. clear(): 清空集合
> 6. size(): 返回集合的大小
> 7. toString(): toString方法
